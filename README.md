# Music and Photo Gallery by The Groove

Brand Introduction: 

"The Groove" is an immersive online experience merging music and visual artistry. As a dynamic music and photo gallery, it invites users to journey through a curated selection of single album cover art while seamlessly immersing themselves in the accompanying melodies. Each slide presents a captivating album cover, inviting exploration and intrigue, with the ability to instantly sample the corresponding song. 

Slogan: Your quick stop for tunes, get your Groove on!

# How to use
This website showcases the photo gallery with a music player.

Running the Website Locally
To run this website locally, follow these steps:
1. Clone this repository to your local machine.
2. Open the index.html using live server viewer in VS Code.
3. View it in your browser of choice
4. Slide through different single covers
5. Control the audio player for play,pause and stop, there is also a volume control

Enjoy!
